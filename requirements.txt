ACORTADOR DE URLs HÍBRIDO
Flask + MySQL + MongoDB
-----------------------

1. DESCRIPCIÓN DEL PROYECTO

Aplicación web para acortar URLs con almacenamiento híbrido:

* MySQL → datos estructurados (URL, código, descripción, fecha)
* MongoDB → almacenamiento de imágenes (preview)

Incluye:

* Interfaz moderna con Bootstrap 5
* Validación de duplicados
* Historial de enlaces
* Redirección automática

---

5. BASE DE DATOS

MySQL:

CREATE DATABASE url_shortener;

USE url_shortener;

CREATE TABLE urls (
id INT AUTO_INCREMENT PRIMARY KEY,
original_url TEXT NOT NULL UNIQUE,
short_code VARCHAR(10) UNIQUE,
description TEXT,
created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP
);

MongoDB:

Base de datos: url_images
Colección: images

Ejemplo documento:

{
"short_code": "AbC123",
"image_url": "https://imagen.com/img.jpg"
}

9. VALIDACIONES IMPLEMENTADAS

* No permite URLs repetidas
* No permite duplicados por descripción (opcional)
* Genera código único
* Manejo de errores con alertas visuales

---

10. POSIBLES ERRORES

Error: run.py no encontrado
→ Verificar ubicación del archivo

Error: MySQL conexión
→ Revisar credenciales en config.py

Error: MongoDB
→ Ejecutar "mongod"

Puerto ocupado
→ Cambiar puerto en run.py
